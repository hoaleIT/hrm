SUPPORT & TROUBLESHOOTING
================================================================================

Issue: Port 3000 already in use
Solution: npm run dev -- -p 3001

Issue: Database connection error
Solution: Check .env.local for correct Supabase URL & API key

Issue: Build errors
Solution: Delete .next folder, run npm install, then npm run dev

Issue: Authentication not working
Solution: Verify .env.local is configured, restart dev server

Issue: Color scheme not showing
Solution: Hard refresh browser (Ctrl+Shift+R), clear browser cache

- Check QUICK_START.md for detailed troubleshooting
- Check browser console (F12) for error messages
